### Tablas

Se pueden permitir la creación de tablas las cuales se visualizan de la misma
manera que se crean pero de forma más limpia

```markdown
| Nombre | Número | Color    |
|--------|--------|----------|
| Juan   | 28     | Verde    |
| Julia  | 11     | Rojo     |
| Pedro  | 4      | Amarillo |
```
De esta manera

| Nombre | Número | Color    |
|--------|--------|----------|
| Juan   | 28     | Verde    |
| Julia  | 11     | Rojo     |
| Pedro  | 4      | Amarillo |

### Checklists

Listas de tareas con una checkbox marcada o desmarcada

```markdown
- [x] Punto marcado
- [ ] Punto no marcado
```

Se vería similar

- [x] Punto marcado
- [ ] Punto no marcado
